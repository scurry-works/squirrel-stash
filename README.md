# Squirrel Stash

A Discord game bot example built with [scurrypy](https://github.com/scurry-works/scurrypy).

## Note
This is a **reference implementation** showing how to build a game bot with scurrypy. 
It's configured for a specific server and is meant to be read and learned from, not cloned and run as-is.

## What it demonstrates
* Slash commands
* Button Interactions
* DB setup
* Addon Patterns
* Pagination
